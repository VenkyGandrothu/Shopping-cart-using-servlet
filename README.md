# Shopping Cart Application

A Java Servlet-based **Shopping Cart Web Application** with JSP front-end and MySQL database.  
This project allows users to **register, login, browse products, and add them to a cart**.

## Features

- User **Registration and Login**
- **Session Management** to maintain login state
- Browse available **products dynamically** from the database
- **Add to Cart** functionality
- Clean, responsive **UI with JSP**
- Navigation for **Cart** and **Logout**

---

## Technologies Used

- **Java Servlet API**
- **JSP (JavaServer Pages)**
- **HTML5, CSS3**
- **MySQL** database
- **Apache Tomcat** server
- **Eclipse IDE** (or any Java IDE)
